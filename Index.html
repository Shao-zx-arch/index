CSS核心概念:背景,盒模型,布局,动画

背景 (Background)

CSS 背景属性可以为元素设置各种漂亮的背景效果。

常用背景属性：

  • background-color - 设置背景颜色
  • background-image - 设置背景图片
  • background-repeat - 控制背景图片是否重复
  • background-position - 设置背景图片位置
  • background-size - 控制背景图片尺寸
  • background-attachment - 控制背景图片是否随页面滚动

CSS 发音提示： "CSS"读作"C-S-S"，是"Cascading Style Sheets"的缩写

盒模型 (Box Model)

CSS 盒模型是网页布局的基础，每个元素都被视为一个矩形盒子。

盒模型相关属性：

  • width/height - 内容区域宽高
  • padding - 内边距
  • border - 边框
  • margin - 外边距
  • box-sizing - 控制盒模型计算方式

动画 (Animation)

CSS 动画可以为网页添加生动的交互效果，提升用户体验。

动画三要素：起始状态、结束状态、持续时间。

常用动画属性：

  • transition - 过渡效果
  • @keyframes - 关键帧动画
  • animation - 动画简写属性
  • transform - 变形效果
